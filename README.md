# CRUD-Lab-3-
CRUD Application

This is a simple CRUD (Create, Read, Update, Delete) web application developed using HTML, CSS, and JavaScript.

Features:
- Add new data (Create)
- Display data (Read)
- Edit existing data (Update)
- Delete data (Delete)

How it works:
- User enters a name in the input field.
- Click "Add" to store the data.
- Data is displayed in a list.
- "Edit" button updates the data.
- "Delete" button removes the data.

Technologies Used:
- HTML
- CSS
- JavaScript

File Name:
- crud.html

How to Run:
- Open the file in any web browser
  OR
- Run using XAMPP:
  localhost/crud/crud.html

Purpose:
This project is created for learning basic web development and understanding CRUD operations.
